# Changelog

本仓库所有重要变更均记录于此文件。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.2.1] - 2026-07-21

### 变更

- **录制配置前置**：按 `S` / 录制按钮先弹出**开录前配置对话框**（格式 MP4/GIF + 画质 高/中/低，默认取上次选择并**持久化**；「开始录制」/「取消」，Esc 关闭），确认后才进入「3S 后开始录制」倒计时；停止后**直接弹系统保存对话框选保存位置**，格式/画质按开录前选择执行、不再重复询问；取消路径选择或保存失败可「重试保存 / 放弃录制」

## [0.2.0] - 2026-07-21

### 新增

- **差值热图对比布局**（A/B 对比第 4 种布局）：`D` 键开关（进入时记住当前布局，再按 D 还原；W/G 仍只循环前三种）；逐像素三通道 max abs diff，**≤容差置黑**，其余归一后过 colormap；四种色带 **inferno / gray / viridis / coolwarm**（coolwarm 为标准发散 LUT：中点近白、两端深蓝深红）+ 容差 0–128；B 图经 canvas 平滑缩放到 A 尺寸对齐，缩放 / 平移不重算，源图 / 容差 / 色带变化才重算（100ms 防抖）
- 差值**配置面板仅 diff 激活时显示**：colormap 下拉 + 容差滑块 / 数值输入，`diffColormap` / `diffTolerance` 持久化到设置
- **显示区录制**（`S` 键 / 工具栏按钮）：3 秒倒计时胶囊（再按 S 取消）→ 红点计时徽标录制 → 3 秒停止倒计时（期间继续录，再按 S 取消停止）→ 保存对话框；格式 **MP4 / GIF** + 画质 高 / 中 / 低 三档；Electron 经 desktopCapturer 采集本窗口并按显示区 rect 镜像裁剪（15fps 上屏），GIF 走 gifenc（10fps）；上限 10 分钟自动停，视图切换自动停止并释放采集

### 修复

- diff colormap 可选列表单一来源化：`DIFF_COLORMAP_VALUES`（inferno / gray / viridis / coolwarm，顺序固定）由 settings 导出，设置校验 / 工具栏下拉 / 冒烟断言三处共用，杜绝列表漂移

### 已知限制

- GIF 为 10fps、最多取录制末尾 30 秒（环形 300 帧缓冲）；视频码率按画质档于录制开始时确定（保存对话框中改画质只影响 GIF 与下次录制）
- MP4 容器依赖 Chromium 的 MediaRecorder 支持，不可用时自动回退 **WebM**（UI 明示）
- diff 差值布局下 ALT 颜色探针不可用（显示的是合成热图而非源图）
- 录制采集整个本窗口再裁剪：录不到系统声音；窗口被最小化 / 遮挡区域画面由系统合成器决定；浏览器模式经 getDisplayMedia 采集（可能包含浏览器 UI），保存 = 触发下载（无法选择位置）

## [0.1.1] - 2026-07-20

### 新增

- 单图 / 对比 / 网格视图中点击文件夹树节点即切回浏览模式并显示该目录网格（原仅切换视野不离开当前视图）；浏览中点击行为不变，Esc 逐层回退不回归
- README 顶部新增「一指禅 · 30 秒上手」区块（对比双击三连、F / Shift+F、Alt 取色、? 快捷键列表）

## [0.1.0] - 2026-07-20

首个正式发布版本。跨平台（Windows / macOS）桌面图片对比浏览器，同一套 React 代码同时可作为网页版运行。

### 新增

#### 浏览 / 文件夹树 / 显示模式

- 缩略图网格懒加载；显示模式四档（大 / 中 / 小图标、列表，持久化）；列表模式点列头排序
- 子文件夹条目（Windows 风文件夹图标 + 预览拼贴 + 计数，双击进入）排在图片前
- 面包屑路径导航（Backspace 返回上级）；文件夹树懒加载 + 祖先链 + 收藏夹（持久化）
- 含子文件夹递归开关（默认关）、格式过滤、名称 / 日期 / 大小排序
- 全选 / 清除选择、勾选导航（全部 / 仅勾选两档）
- 打开文件夹「选择即打开」：Electron 自绘对话框（快捷入口 + 子目录单击选中 / 双击进入 + 右侧本层预览：子文件夹条目与本层图片缩略图，不递归）；系统对话框选中文件 = 打开所在文件夹并定位

#### 单图

- 适应窗口 / 100%、滚轮锚点缩放、拖拽平移、R/L 旋转 90°（仅视图层）
- 双缓冲无缝切图（canvas 重采样 + 统一解码缓存，消除黑闪）

#### A/B 对比与多图网格

- 三种对比布局：**划变**（同区域对齐 + 可拖分割线）/ **并排**（可拖比例 + 同步开关）/ **叠化**（onion-skin 透明度，可换上下层），W/G 循环
- Tab 切激活侧、X 交换、N 下一对；槽位导航优先跳过另一槽占据项，无其他项时允许同图
- 多图网格对比：勾选 ≥3 张进入（最多 9 张），自动 / 手动布局，同步 / 独立两档，数字键选格，N 下一组

#### 解码缓存与缩放算法

- 会话级字节预算 LRU 解码缓存（1GB）、在显 pin 保护、预取（对比集合 / 单图 ±1）、调试日志
- 缩放算法六档：自动 / 邻近 / **BIFant / 双线性 / 双立方 Catmull-Rom / Lanczos-3**（软件精确重采样：浏览平滑预览，停手 ~150ms 后 CPU 可分离卷积精确重绘；切算法零解码）

#### 直方图 / EXIF / 颜色探针

- 信息浮层：基本信息 + EXIF（exifr，按图缓存）
- 纯亮度直方图（FastStone 风格，Rec.709 单通道 256 bin，悬停显示 bin 像素数与占比）
- ALT 颜色探针：原图坐标 + RGB(A) 浮签（经 transform 逆映射）；ALT+单击记入侧栏取样列表（≤10 条）

#### 文件操作

- 右键文件操作：复制 / 粘贴 / 新建文件夹 / 删除（Electron 删除进回收站）
- 系统拖放文件 / 文件夹递归复制到当前目录（带投放指示层）

#### 全屏体系

- **双击三层交互链**：对比 / 网格双击某格 → 单格控件全屏 → 物理全屏 → 循环切换显示源（对比 A↔B / 网格下一格，槽位与格组内容不变）；单图双击 = 适应 ↔ 100%
- F = 当前格控件全屏开关；Shift+F = 当前视图整体物理全屏
- 物理全屏 = 真全屏：卸载工具栏 / 侧栏 / 胶片条，只留图像 + 浮层 + 悬浮迷你条，退出自动恢复

#### CLI 与集成

- CLI 参数解析（`twinview <目录|文件> [--compare A B] [--layout side|wipe|overlay] [--theme dark|light] [--recursive]`）+ 单实例转发（cli-open IPC）

#### 主题

- 暗色 / 亮色 / 跟随系统三档（持久化；Electron 窗口背景同步，首帧防闪）

#### CI / 发布

- GitHub Actions 流水线：verify（类型检查 + 网页版构建）→ Windows NSIS x64 / macOS arm64 DMG 矩阵打包
- main 分支滚动 nightly 预发布；`v*` tag 触发正式发布（本版本起）

### 修复

- 控件全屏（双击链 L1）下 X 切显示源到第 2 张后 ←/→ 失效：navigate 改为作用于**当前显示的槽/格**（对比全屏槽位、网格全屏格索引），X 交换行为不回归
- BIFant 改真面积平均（覆盖长度权重），消除 ≥78% 缩放 / 放大的周期性横竖线
- macOS DMG 改 ad-hoc 签名，修复 arm64「已损坏，无法打开」（首次打开仍需「右键 → 打开」）
- 对比模式 X 交换失效修复；仅勾选导航允许槽位重复（移除误报提示）
